# Entertain Bar POS

## Objective

A web-based POS system for a small entertain bar focused on speed,
accuracy, and ease of use.

## Tech Stack

-   Next.js
-   TypeScript
-   Tailwind CSS
-   Prisma ORM
-   PostgreSQL (Neon)
-   Vercel

## Roles

-   ADMIN: Reports, configuration, product management.
-   CASHIER: Open/close bills, payments, expenses.
-   STAFF: Assist with taking orders and viewing bill/history.

## Main Workflow

1.  Open Bill
2.  Add Items
3.  Review Bill
4.  Receive Payment
5.  Close Bill
6.  Daily Closing Report

## Development Principles

-   Analyze before coding.
-   Design database first.
-   Build Prisma schema before backend/frontend.
